## [0.1.0] - 2025-12-24

### Added
- Initial package structure with src-layout (core, fs, ops, undo modules)
- PEP 621 compliant pyproject.toml with hatchling build backend and dynamic versioning
- MIT License
- Pre-commit hooks with ruff linter/formatter
- Renovate bot for automatic dependency updates
- GitHub Actions workflows: build, tests, docs deployment, PyPI/GHCR publishing
- Dockerfile (Python 3.12-slim)
- MkDocs documentation with Material theme

See [#1](https://github.com/Louis-Pujol/ordo-filesystem/pull/1) for details.
