## [0.1.0] - 2025-12-24

### Added
- Initial package structure with src-layout
- Empty modules for core, fs, ops, and undo functionality
- PEP 621 compliant pyproject.toml with Python 3.11+ requirement
- MIT License
- Hatchling build backend
- Pre-commit hooks configuration with ruff linter and formatter
- Renovate bot configuration for automatic dependency updates
- GitHub Actions workflows for build and tests
- Dockerfile with Python 3.12-slim base image
- MkDocs documentation with Material theme
- API documentation structure for all modules
- Dynamic versioning from __init__.py
